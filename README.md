## Tester le projet

Instaler les dépendances avec 
```
npm install
```

Il est nécessaire d'avoir un serveur mongoDB, en ligne ou local
Vous pouvez changer ces variables dans le .env 
```
MONGO_HOST
MONGO_PORT
MONGO_DB 
EMAIL 
EMAIL_PASS 
BASE_HREF = "http://localhost:5173"
STRIPE_PUBLIC_KEY 
STRIPE_SECRET_KEY 
```

Pour démarer le projet : 
```
npm run dev
```
